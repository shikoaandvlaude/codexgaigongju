## 提交注意事项

1. **通用型漏洞**才能报 CNVD（单个站点的洞报 SRC 不报 CNVD）
2. 影响面越大评分越高 → 用 FOFA 统计受影响资产数写进报告
3. 提交后等待审核（通常 3-15 个工作日）
4. 审核通过后会分配 CNVD 编号
5. 可以同时提交 CVE（英文版报告给 MITRE）= 一洞两吃

## 双提交流程

```
发现通用漏洞
├── 中文报告 → 提交 CNVD → 获得 CNVD-20XX-XXXXX
└── 英文报告 → 提交 MITRE → 获得 CVE-20XX-XXXXX
    (两个体系互不冲突，可以同时提交)
```
