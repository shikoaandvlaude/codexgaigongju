# Bai-codeagent

先看 `TOKEN_MODE.md`。它是给日常使用准备的短版入口，目的是少读文档、少烧 token。

`know.md` 和 `readmefirst.md` 只在需要完整参考时再打开。

## 核心命令

```bash
/recon target.com
/hunt target.com
/validate
/report
/autopilot target.com --normal
```

## 结构

- `server.js` - Web 面板
- `redops/` - RedOps Agent
- `claude-hunt/` - 自动化挖洞工具
- `know.md` - 完整知识库
- `TOKEN_MODE.md` - 省 token 运行指南
